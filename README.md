# Podcaster

## Backend

### Installation
```
cd ~/path/to/podcast-backend/api
yarn install
```

```
cd ~/path/to/podcast-backend/mailService
yarn install
```

### Development Start

`API`

Open the terminal and run:
```
cd ~/path/to/podcast-backend/api
yarn run watch
```

Then in a second terminal run:
```
cd ~/path/to/podcast-backend/api
yarn run start
```

`Mail Service`
```
cd ~/path/to/podcast-backend/mailService
yarn run watch
```

Then in a second terminal run:
```
cd ~/path/to/podcast-backend/mailService
yarn run start
```
